# Python-Scraper

The purpose of this project is to build scrapers to collect movie theather information from the top 4 natonal theater chains.

## Theater Chains
* AMC
* Regal
* Cinemark
* Cineplex

## Information to Collect
* Theater Addresses
* Movie Titles
* ShowTimes
* MPAA Ratings
* Run Times
* Screen Types (standard, IMAX, 3D, ect.)
